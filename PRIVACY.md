# PRIVACY

## 単一用途

### 単一用途の説明

```
SYNCROOMのプレイヤーズサイト(https://syncroom.yamaha.com/play/)に機能を追加します。
```

## 権限が必要な理由

#### storage が必要な理由

```
お気に入りメンバー情報、空き通知情報を保存するため。
```

#### tabs が必要な理由

```
アイコンや通知をクリックした際に新規タブでページを開くため。
```

#### notifications が必要な理由

```
満室の空き情報を通知するため。
```

#### ホスト権限 が必要な理由

```
APIから部屋情報を取得しているため。
```
