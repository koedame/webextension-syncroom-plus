# SYNCROOM-PLUS

[SYNCROOM のプレイヤーズサイト](https://syncroom.yamaha.com/play/)に情報を追加する拡張機能です。

## 機能

- 👤 メンバー表示機能
- ⏳ 残り時間表示機能
- ⭐ お気に入り機能
- 🔍 絞り込み機能
- 🚫 満室表示

## 対応ブラウザ

- ✅ Google Chrome
- ✅ Mozilla Firefox
- ✅ Opera
- ✅ Microsoft Edge
